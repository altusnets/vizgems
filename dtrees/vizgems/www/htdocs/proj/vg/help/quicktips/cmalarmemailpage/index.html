10.4     Alarm Email Filter File

 

This file contains records that direct the system to send out email notifications for alarms that match the selection parameter.

 

Figure  10.3 – Alarm Email Filter File

 

Key field descriptions are provided in the following table.

 

Table  10.2 – Alarm Email Fields

Key Fields

Descriptions

Asset Name

full name of an asset or a regular expression describing multiple assets:e.g., b01234.* - common regular expressions meta characters allowed: $ ^ [ ].*

Alarm ID

full alarm id or a regular expression describing multiple alarms - only alarms known to the correlation system have alarm ids

Message Text

 

full message text or a sub string of it - no regular expressions are recognized except for the use of .*: Threshold.*cycles

Ticket Mode

select a specific mode or Any to match all modes

Severity

select a specific severity or Any to match all severities

Start Time

(HH:MM) of this action

End Time

(HH:MM) of this action

Start Date

(YYYY.MM.DD) of this action. Note: Please use present date and future date with required format.

End Date

(YYYY.MM.DD) of this action. Note: Please use present date and future date with required format, or set as ‘indefinitely’.

Repeat

for daily, weekly, and monthly modes, the start and end times specify the initial time interval

From Address

from email address

To Addresses

list of To email addresses

Email Style

the style of email message - HTML: tabular layout, Text: simple textual layout, SMS: short textual layout appropriate for cell phone text messages

Email Subject

the subject line of the email message

User Account

the website account that established this email rule

Bulk Insert File

a file with records to insert to the main file - must be in the same format as the main file
