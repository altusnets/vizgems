10.13     Type File

 

This file contains a list of asset type records which is used within the system and is similar as a database table. This file is used to match unique asset type IDs with asset short type names and asset full type names.

 

Figure  10.12 – Type File

 

Key field descriptions are provided in the following table.

 

Table  10.11 – Type File Key Fields

Key Fields

Description

Type ID

the unique id for the type, e.g., idc--web

Short Type Name

the short type name, e.g., WEB

Full Type Name

the full type name / description, e.g., WEB - WEB SERVER
